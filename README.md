# Budget Tracker

Personal finance and budget tracking application built with Kotlin.

## Features (MVP)
- Add income and expense transactions
- List all transactions
- Calculate current balance

## Tech stack
- Kotlin/JVM
- Gradle (Kotlin DSL)
- JUnit

## Project structure
- domain — core business models and rules
- application — use cases and ports
- adapters — CLI, persistence, API (planned)

## Build
```bash
./gradlew build
